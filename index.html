Proyectos destacados

Trabajos y desarrollos que demuestran mis habilidades técnicas

ACorsaHostingES

Servicio de hosting especializado en servidores dedicados para Assetto Corsa, un simulador de conducción profesional. La plataforma ofrece infraestructura optimizada para comunidades de simracing que organizan campeonatos y eventos competitivos online.

El proyecto incluye gestión de servidores Linux, configuración de servicios web, administración de bases de datos para almacenar tiempos y clasificaciones, y desarrollo de APIs para integración con sistemas de timing en tiempo real.

Linux Server Web Hosting API REST Gestión de Comunidades Bases de Datos Server Hosting

TaskFlow Manager

Aplicación web de gestión de tareas y proyectos académicos diseñada para estudiantes y equipos de trabajo. Permite organizar entregas, dividir proyectos en fases, asignar prioridades y visualizar el progreso mediante tableros Kanban.

El sistema incluye autenticación de usuarios, gestión de equipos colaborativos, sistema de notificaciones para fechas límite y exportación de reportes en formato PDF. Desarrollado con enfoque en usabilidad y diseño responsive.

HTML5 CSS3 JavaScript Responsive Design UX/UI

EcoMetrics Dashboard

Panel de visualización de datos ambientales para centros educativos y pequeñas organizaciones. Muestra métricas de consumo energético, gestión de residuos, emisiones de CO₂ y objetivos de sostenibilidad mediante gráficos interactivos.

El proyecto implementa conexión con sensores IoT para monitorización en tiempo real, almacenamiento histórico de datos, generación de informes automatizados y sistema de alertas configurables. Incluye administración multiusuario con diferentes niveles de acceso.

Data Visualization IoT Integration SQL Database Responsive Web Analytics
